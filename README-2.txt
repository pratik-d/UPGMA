GENUTZTE PROGRAMMIERSPRACHE UND SYSTEM:
		
			Betriebssystem und Entwicklungsumgebumg:
			
			- verwendetes Betriebssystem : 	Windows 8.1 Professional
			- Entwicklungumgebung : Notepad++

AUSFÜHREN DES PROGRAMMS:

			Bevor das Programm ausgeführt werden kann sind folgende Schritte nötig:
			
					1. Öffnen Sie die Datei AlBi1.py mit ihrem Texteditor
					2. Passen Sie die Variable "Pfad" (in der 332. Zeile) entsprechend dem Quellverzeichnis ihrer Fasta-Datei ab.
					( Bsp: "C:\\User\\Username\\AlBi\\Projekt1\\" )
					3. Falls ihre Datei nicht dem default-name entspricht ("praktikum1_sequences.fasta") ändern Sie den Eintrag in der Zeile 12.
					
			Ausführung:
					
					Für das Ausführen der AlBi1.py Datei, führen Sie (unter Windows) einfach einen Doppelklick auf ihr aus.
					
			Resultat:
					Als Resultat erhalten Sie phylogenetischen Bäume, sowohl für den ursprunglichen Sequenzen als auch für die mutierte Möglichkeiten.
